## Dependencies
```
python3 -m pip install -U pygame --user
```

## How to play
1. start the program
```
python3 main.py
```
2. The game window will pop up.
![Screenshot](./pic/demo.jpg)

3. Control
- orrange cat: Luca
    - w: up
    - a: left
    - s: down
    - d: right
    - bullet: left control
- white cat: Lumi
    - up: up
    - left: left
    - down: down
    - right: right
    - bullet: right control

4.  Have fun with your family or friend.
